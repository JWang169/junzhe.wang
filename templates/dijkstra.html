Introduction

Dijstra's Algorithm is aim to solve single source shortest path problem.

The key point of Dijstra and Bellman is Relexation: 

    if d[u] + c(u, v) < d[v]:
      d[v] = d[u] + c(u, v)
Steps

  1. Select a start vertex.
  2. Compute the distance to its connected vertices.
  3. Set the connected node with the shortest distance as the start point, and updates its neighbors.
  4. After all the vertices are updated as the start point, end it.

Analysis
  • Time complexity is O(n^2). For each vertex as the start point, we need to update all its neighbors. The worst scenario is that all vertices have n neighbors, where n is the total amount of vertices.
  • Dijkstra can work on both directed and non directed graphs.
  • If there is a negtive edge in the graph, it might not be correct.
