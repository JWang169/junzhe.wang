KMP

KMP(Knuth-Morris-Pratt) is for pattern matching, including prefix, suffix, substring, etc. It can search for the pattern in linear time. If we do brute force, the time complexity will be O(m*n). If we do KMP, it will be O(m + n). The key step is to build the table for the pattern.

The failure function

Take inorder traversal for example.

  1. Initialize the root as current node.
  2. While current is not None, check if current has a left node.
  3. If current does not have a left child, we can print this node and then update it to point to its right node.
  4. If current has a left child, we find the most right child of current as the predecessor, and make the right child of the predecessor to current. (This is to link the most right child to root. Since the most right child is the last node we visit on the left side of the current root, we need to go back to root after visiting the last node on the left. )
  5. Update current to its left node.

Code

InOrder

def morris(root):
    result = []
    while root:
        if not root.left:
            result.append(root.val)
            root = root.right
        else:
            predecessor = root.left
            while predecessor.right and predecessor.right != root:
                predecessor = predecessor.right
            if not predecessor.right:
                predecessor.right = root
                root = root.left
            # predecessor.right == current, meaning we're going back to the root
            else:
                predecessor.right = None
                result.append(root.val)
                root = root.right
                
    return result

PreOrder

def preorderTraversal(root):
    r = []
    while root:
        if not root.left:
            r.append(root.val)
            root = root.right
        else:
            pre = root.left
            while pre.right and pre.right != root:
                pre = pre.right

            if not pre.right:
                r.append(root.val)
                pre.right = root
                root = root.left
            else:
                root = root.right
    return r

PostOrder

def postOrder(root):
    result = []
    while root:
        if root.right:
            prev = root.right
            while prev.left and prev.left != root:
                prev = prev.left
                
            if prev.left:
                root = root.left
            else:
                result.append(root.val)
                prev.left = root
                root = root.right
        else:
            result.append(root.val)
            root = root.left
    return result[::-1]
